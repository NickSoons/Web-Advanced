De splice methode - Array Javascript

Wat is deze methode?

Deze methode is voor bepaalde elementen in uw array te voegen op de bepaalde plaatsen.

Originele Array

In de onderstaande array word de sport pingpong toegevoegd vanaf plaats 2 en er word niets verwijderd

Ook kan he bepaalde items weghalen en verplaatsen op bepaalde plaatsen in een array

Als laatste kan je ook items verwijderen
